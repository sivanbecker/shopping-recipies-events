# Security Policy

## Supported Versions

This is a personal project. Only the latest version receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please open a [GitHub issue](https://github.com/sivanbecker/shopping-recipies-events/issues) or contact the maintainer directly via GitHub.

Since this is a personal project, response times may vary. You can generally expect an acknowledgement within a few days. If the vulnerability is accepted, a fix will be prioritized based on severity. If declined, an explanation will be provided.
